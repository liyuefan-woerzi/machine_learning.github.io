Dimensionality Problem

When working with data for a given problem we can have an arbitrary number of features. Some of those features might be relevant or not to our problem. Dimensionality describes the number of features we have, having many features and hence a high dimensionality doesn't always contribute to a better prediction. This is called the dimensionality problem .

Adding more features requires more sample points to avoid overfitting.

Ignoring irrelevant features can improve accuracy

Techniques to Deal with high dimensional problems

  • Supervised Feature Selection

    Select features to use in our model by hand, ignore irrelevant features, combine features together.

    • We can use accuracy score to determine which features to use
    • We can use algorithms such as decision trees to find most informative features (e.g. Entropy metric)

    Techniques involve:

    • Forward Selection: Start with empty set of features and add more features incrementally while evaluating your accuracy

    • Backward Selection: Start with all features available and remove one by one

      These techniques assume that the features are independent which is rarely the case.

  • Unsupervised Feature Selection

    Principal Component Analysis

    • Find a new feature space

    • Assumes that most important features are in the projection with the greatest variance

      Example of reducing 2d to 1d

Principal Component Analysis

The goal is to find vectors or components that approximately describe the data the same way.

For PCA on the same dimension we find the same number of components as the dimensions we are working with. It is relative trivial problem.

Given a dataset: 

xxxxxxxxxx
from numpy import array
from numpy import mean
from numpy import cov
from numpy.linalg import eig
# define a matrix
A = array([[1, 2], [3, 4], [5, 6]])
print(A)
# calculate the mean of each column
M = mean(A.T, axis=1)
print(M)
# center columns by subtracting column means
C = A - M
print(C)
# calculate covariance matrix of centered matrix
V = cov(C.T)
print(V)
# eigendecomposition of covariance matrix
values, vectors = eig(V)
print(vectors)
print(values)
# project data
P = vectors.T.dot(C.T)
print(P.T)

Dimensionality Reduction with PCA

If we want to reduce the dimensions of our data using PCA again we find all the component/vectors for each dimension and select the vectors with the highest variance.

Comparing Classifier on PCA Data 

xxxxxxxxxx
from sklearn.datasets import load_digits
from sklearn.decomposition import PCA
import pandas as pd
from sklearn.linear_model import LogisticRegression
from sklearn.model_selection import train_test_split
plt.rcParams['figure.figsize']=(10,5)
data = load_digits()
X = data.data
y = data.target
pca = PCA(n_components=2)
X_pca=pca.fit_transform(X)
​
X_train,X_test,y_train,y_test=train_test_split(X,y)
clf=LogisticRegression().fit(X_train,y_train)
print("Clean data accuracy score: %f"%clf.score(X_test,y_test))
​
​
X_train,X_test,y_train,y_test=train_test_split(X_pca,y)
clf=LogisticRegression().fit(X_train,y_train)
print("PCA Score 2 componenets: %f"%clf.score(X_test,y_test))
​
​
​
pca = PCA(n_components=10)
X_pca=pca.fit_transform(X)
​
X_train,X_test,y_train,y_test=train_test_split(X,y)
clf=LogisticRegression().fit(X_train,y_train)
print("PCA Score 10 components: %f"%clf.score(X_test,y_test))
​
​
plt.scatter(X_pca[:,0],X_pca[:,1],c=y)
plt.colorbar();
plt.show()

TSNE

t-distributed Stochastic Neighbor Embedding. t-SNE is a tool to visualize high-dimensional data. Works in a similar way as PCA, but often produces far more desirable results. It can also take a lot of time to run so it is suggested we first run PCA on high dimensional data and then pass it into t-SNE to visualize it. 

xxxxxxxxxx
from sklearn.manifold import TSNE
​
X_tsne = TSNE(n_components=2).fit_transform(X)
X_train,X_test,y_train,y_test=train_test_split(X_tsne,y)
clf=LogisticRegression().fit(X_train,y_train)
print("TSNE Score 2 components: %f"%clf.score(X_test,y_test))
