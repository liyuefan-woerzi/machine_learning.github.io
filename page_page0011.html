Unsupervised Learning

It is a machine learning technique used on unlabeled data. This technique can be used to existing labelled data to find new patterns and to unstructured/unlabelled data to find potential patterns. Clustering and Dimensionality Reduction (PCA) are the problems we will look within the unsupervised learning domain.

Clustering

The goal of a clustering algorithm is to find commonalities between multiple data-points. This can help for existing labelled data points to find features which have commonalities between them and can be used as stronger indicators in other supervised tasks. For data that is not labelled we can use the very same technique to assign a label. There are many algorithms within this problem space that solve the same problem differently or for different types of data. One of the more straightforward methods is K-Means clustering.

K-Means

K-Means clustering is a method by which we take N-Data points and divide them into K clusters (Hence the name k-means clustering), with .

The goal of the algorithm is separate the datapoints such that the distance between the center of the clusters are minimized. We call the center of the clusters centroids .

Pseudo Code 

xxxxxxxxxx
Specify N, the number of clusters
Randomly assign each point to a cluster
​
do
  Calculate the center of clusters using a Distance Metric (e.g. Euclidean Distance), centroid
  Correct the predictions by assigning each point to the cluster with the closest centroid
  Recalculate the centroids
reapeat if centroid changes

  • We reassign the points to the cluster closest to them using the Euclidean Distance metric and we recalculate the centroids. 

    xxxxxxxxxx
    cluster_0=X[clusters==0]
    cluster_1=X[clusters==1]
    cluster_2=X[clusters==2]
    # axis=0 adds the points vertically to seperate between different features
    center_0=cluster_0.sum(axis=0)/cluster_0.shape[0]
    center_1=cluster_1.sum(axis=0)/cluster_1.shape[0]
    center_2=cluster_2.sum(axis=0)/cluster_2.shape[0]
    ​
    # Euclidean distance: sqrt ( (cluster center - x)^2 )
    d_c_0=np.sqrt(((center_0-X)**2).sum(axis=1))
    d_c_1=np.sqrt(((center_1-X)**2).sum(axis=1))
    d_c_2=np.sqrt(((center_2-X)**2).sum(axis=1))
    d_c=np.array(list(zip(d_c_0,d_c_1,d_c_2)))
    clusters=np.argmin(d_c,axis=1)
    plot_clusters(X,clusters)

K-Means Drawbacks

  • It makes assumptions on the data

    • Shape assumption: Because K-means uses a distance metric, the clusters will have spherical shape

  • Bad K parameter : if a bad K-parameter is provided the clusters will be unevenly distributed and bad clusters can be formed.

    • Unequal Variance : More spread out clusters (higher variance) can be mislabelled due to the distance metric claiming them closer to the centroid of a nearby cluster.
    • Size of cluster: It doesn't affect the algorithm at all because the distance metric is irrespective of the cluster size.

Scaled Features

We discussed in previous classes how unscaled features can cause a real problem

Something important to keep in mind is that if our data is NOT scaled, one term in the summation of the distance will over-take all other terms in terms of size and be the only determining factor for the distance metric.

e.g. if one feature takes values between 1 and 10000 and another feature takes values between 0 and 1 the feature with the range 1-10000 will be the only determining factor for how far the data is spread.

Hierarchical Clustering

Other techniques overcome the drawbacks of K-Means by making different assumptions. Hierarchical clustering for example tries to find hierarchies of datapoints in such a way that we can form many clusters. Two approaches exist, either we start top-down, we start from 1 large cluster and split into smaller clusters or we start bottom-up, we start from each point being an individual cluster and we move up.

Definition of Closeness and Linkage

We need to define some metric to measure how close two clusters are from each other. The closer the better. A Euclidean metric can be used similar to K-Means.

  • Ward minimizes the sum of squared differences within all clusters. It is a variance-minimizing approach and in this sense is similar to the k-means objective function but tackled with an agglomerative hierarchical approach.
  • Maximum or complete linkage minimizes the maximum distance between observations of pairs of clusters.
  • Average linkage minimizes the average of the distances between all observations of pairs of clusters.
  • Single linkage minimizes the distance between the closest observations of pairs of clusters.
